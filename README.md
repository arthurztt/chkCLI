# chkCLI - Checagem de endpoints HTTP

O **chk** é uma ferramenta que permite realizar checagens em endpoints HTTP via comandos no Terminal por meio de requisições `GET`. Conseguindo receber os seguintes parâmetros : Status Code da requisição `GET`, Tempo de Retorno do comando em Milisegundos(ms), URL do endpoint requisitado.

## Requisitos

- Go(v1.26+) - [Download da Linguagem Go](https://go.dev/dl/)
- Biblioteca de Comandos CLI do Go - `go get github.com/spf13/cobra`

## Estrutura do Projeto

```
    chkCLI/
      │ main.go
      │ go.mod
      │ go.sum
      └── cmd/
        └── root.go
```
